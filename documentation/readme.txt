NovaLanding - Plantilla Premium para Tienda
---------------------------------------
Una plantilla HTML espectacular con Bootstrap 5 para mostrar productos.

Instrucciones:
1. Descomprime NovaLanding.zip.
2. Abre index.html en un navegador.
3. Personaliza HTML y assets/css/style.css.

Características:
- Responsive con Bootstrap 5.
- Páginas: Tienda (tendencias, pedidos, ofertas, info, marcas), Nosotros, Contacto (WhatsApp).
- Animaciones detallistas y diseño profesional.
- Footer con redes sociales y fecha de actualización.

- Bootstrap 5 (CDN).
- Font Awesome 5 (CDN para íconos).
- Imágenes: Unsplash (libres). Agrega tus propios logos en /images/.
- Fuente: Poppins (Google Fonts).
- Colores: Negro (#0D0D0D), Gris (#2F2F2F), Azul (#4A90E2), Dorado (#D4A017).
